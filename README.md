# Grip Movie app

영화를 검색하고 및 즐겨찾기를 등록할 수 있는 무비 앱 입니다.

<br />

## 👬 **팀원**

- 개인 프로젝트

<br>

## 📅 **개발 기간**

- 2022년 05월 11일 ~ 2022년 05월 15일

<br />

## 🔧 **기술스택**

- Typescript, React, SASS, recoil

<br />

## 💻 **설치 및 실행 방법**

1. yarn 설치

```
 npm i yarn
```

2. repository 클론

```
git clone https://github.com/Yu-jae-min/POB_Grip
```

3. dependencies 설치

```
yarn install
```

4. 실행

```
yarn start
```
